# 오픈소스프로그래밍 프로젝트

<br></br>

## 주제명 : 여행 계획 도우미 음성 비서

## 팀원 구성 및 역할
- 팀장 : 국립한밭대학교 컴퓨터공학과 윤나은, AI 파트 담당
- 팀원 : 국립한밭대학교 컴퓨터공학과 김연주, AI 파트 담당
- 팀원 : 국립한밭대학교 컴퓨터공학과 설희진, 프론트엔드 개발, 디자인
- 팀원 : 국립한밭대학교 컴퓨터공학과 육종범, 백엔드 개발
- 팀원 : 국립한밭대학교 컴퓨터공학과 정광성, AI 파트 담당

## 프로젝트 개요
사용자의 음성 명령을 통해 예행 계획을 쉽게 세울 수 있도록 목적지, 숙소, 교통편을 추천하는 시스템

## 주요 기능
- 음성인식 : 사용자 음성 명령을 텍스트로 변환하여 처리
- 목적지 추천 : 사용자의 선호도와 일정을 반영한 여행지, 숙소, 교통편 추천
- 조회 기능 : 추천된 숙소, 교통편 예약 및 조회 기능

## 시스템 워크플로우
- 사용자가 목적지, 일정, 선호도를 음성/텍스트로 입력
- 프론트엔드에서 데이터를 백엔드로 JSON 형태로 전송
- 백엔드는 공공데이터 포털에서 추천에 필요한 숙소 정보 조회, AI 파트는 사용자 맞춤형 정보 반환
- 추천데이터를 JSON 형태로 프론트에 반환
- 추천 결과를 UI 및 음성으로 사용자에게 안내
